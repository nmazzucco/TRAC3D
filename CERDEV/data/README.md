This folder contains the input datasets required to run `CERDEV.R`.

## Files

- `Supplementary Material_S1.xlsx`  
  Input dataset used as the main training dataset in the analytical workflow.

- `Supplementary Material_S2.xlsx`  
  Input dataset used as the test/experimental dataset in the analytical workflow.

## Notes

These files are read directly by the script through relative paths:

- `data/Supplementary Material_S1.xlsx`
- `data/Supplementary Material_S2.xlsx`

To run the script successfully, both files must remain in this folder and keep the same filenames.
