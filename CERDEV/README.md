# CERDEV

This folder contains the R script, input files, and output files used for the analyses presented in the manuscript:
Mazzucco & Ibáñez 2026: "A New Method for Estimating Prehistoric Agricultural Productivity through the Microtextural Analysis of Harvesting Tools"

## Contents

- `CERDEV.R`  
  Main R script used to run the full analytical workflow.

- `CERDEV_report.Rmd`: R Markdown report describing the workflow and outputs of the analysis.

- `data/`  
  Input files required to run the script:
  - `Supplementary Material_S1.xlsx`
  - `Supplementary Material_S2.xlsx`

- `output/`  
  Output files generated by the script, including figures, tables, and model objects.  
  A detailed description of these files is provided in `output/README.md`.

## Input files

The script reads the following files from the `data/` folder using relative paths:

- `data/Supplementary Material_S1.xlsx`
- `data/Supplementary Material_S2.xlsx`

These are the input datasets used in the analysis.

## Output files

All output generated by the script is saved automatically in the `output/` folder.

These include:
- processed `.csv` tables
- figure files (`.png`, `.jpg`, `.jpeg`)
- fitted model objects (`.RDS`)

A file-by-file description is available in:

- `output/README.md`

## How to run

1. Download or clone the repository.
2. Open the `CERDEV` folder as the working directory in R or RStudio.
3. Make sure the required R packages are installed.
4. Run `CERDEV.R`.

The script uses relative paths, so it expects the following structure:

```text
CERDEV/
├── CERDEV.R
├── README.md
├── data/
│   ├── Supplementary Material_S1.xlsx
│   └── Supplementary Material_S2.xlsx
└── output/
